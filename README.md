
  # KEKAR JAYA

  This is a code bundle for KEKAR JAYA. The original project is available at https://www.figma.com/design/PCHJ6Au0dA3g4gz3I0jaGV/KEKAR-JAYA.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Production env (optional)

  To ensure WhatsApp messages always include a public product image URL, set:

  - `VITE_SITE_URL=https://your-project-name.vercel.app`

  You can copy `.env.example` to `.env` for local setup, and add the same key in Vercel Project Settings → Environment Variables.
